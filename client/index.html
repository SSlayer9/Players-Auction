Welcome to Players Auction!

This is an application for implementing auctions of players on-line. These kind of autions are usually necesary for team tornaments on video games, where there are a few teams with a lot of players joined. Teams get the same ficticial money and they nominate and bid in order to get the best players.

The coin used for ficticial money in this application is K (kilocredits). It can only vary with intervals of 0.5. This means, for example, you cannot offer 5.1K. You must offer integer values or decimals multiple of 0.5K.

This is a server which can host multiple auctions, one per room. In order to join an auction click the buttons placed on the middle column. If you are an administrator, you can create rooms typing the command /makeroom Room Name

In order to get a full commands list type the commands /help and /adminhelp

This application is Open Source. Players Auction is distributted in terms of MIT License.

Basic Commands

  • /login - Opens the login dialog.
  • /register - Opens the regiister dialog. Use it to register an account.
  • /rename <name> - Changes your name but not your id.
  • /changepassword - Opens a dialog for changing your password.

  • /join <room> - Joins a room.
  • /leave <room> - Leaves a room.

  • /nominate <player> - Nominates a player.
  • /pass - Passes the turn.
  • /bid <money> - Bids for a player.

  • /manager <user> - Promotes to room manager.
  • /moderator <user> - Promotes to room moderator.
  • /deauth <user> - Demotes form room manager / moderator.

  • /private - Makes the room private-access only.
  • /public - Makes the room public.
  • /ban <user> - Bans an user from the room.
  • /unban <user> - Unbans an user.
  • /modchat <0|1|2> - Sets the modchat level.

  • /finalizebid - Forces a bid to finalize
  • /teamname <new-team-name> - Fixes a team name (for uppercase/lowercase).

Administration Commands

  • /deregister <user> - Removes an account. Much caution!
  • /admin <user> - Promotes to administrator.
  • /deadmin <user> - Demotes from administrator.

  • /makeroom <name> - Creates an auction room.
  • /deleteroom <room> - Deletes a room. Much Caution!.
  • /roomname <name> - Changes the name of the room.
